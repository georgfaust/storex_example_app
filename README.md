# Storex Example App

```
$ mix deps.get
```

There are vanilla and solid examples (select an app in `config/config.exs`).

For vanilla:

```
assets/vanilla$ npm i
assets/vanilla$ sh ./build.sh
```

For solid:

```
assets/solid$ npm i
assets/solid$ node build.js
```

Then run the server:

```
$ mix run --no-halt
```
